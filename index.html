Welcome to OWAPI.

OWAPI stands for Open Web Application Programming Interface. It allows you to create RESTful APIs within seconds. The current status is experimental!

Prerequisites

To be able to run OWAPI you have to install:

  • NodeJS
  • npm
  • bower
  • MongoDB

Installation

  1. Get the newest version from GitHub
  2. Run npm install
  3. Run bower install
  4. Change the file config/env/development.example.js to config/env/development.js
  5. Optional: Modify the config/env/development.js file to fit your environment
  6. Run node server in the OWAPI root directory
